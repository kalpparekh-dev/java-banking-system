# 🏦 Java Banking System

A simple CLI-based Banking System built using Java.

---

## 🚀 Features

* Create multiple bank accounts
* Deposit money
* Withdraw money
* Check account balance
* Unique account number system
* Search account functionality
* Exception handling (prevents crashes)

---

## 🧠 Concepts Used

* Object-Oriented Programming (OOP)
* Classes & Objects
* ArrayList
* Loops & Conditions
* Exception Handling (try-catch)
* User Input Handling (Scanner)

---

## 💻 How to Run

1. Compile:
   javac BankApp.java

2. Run:
   java BankApp

---

## 📂 Files

* BankAccount.java → Account class
* BankApp.java → Main program
